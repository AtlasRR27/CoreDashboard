CoreDashboard

CoreDashboard is a premium Android dashboard designed to reveal your device’s core metrics in real time. With a clean overlay aesthetic and lightweight architecture, it delivers essential insights without clutter or unnecessary dependencies.


✨ Features
- Real‑time overlay with CPU frequency, RAM usage, battery level & temperature.  
- WiFi details including SSID and signal strength.  
- GPS activity detection.  
- Foreground service keeps the dashboard always available.  
- Proprietary build with obfuscated APK for enhanced security.  


🔒 Privacy & Permissions
CoreDashboard is proprietary software. Source code is not publicly available.  
The app requests only the permissions required for its functionality:

- FOREGROUND_SERVICE → Keeps the overlay active.  
- SYSTEMALERTWINDOW → Displays overlays globally.  
- BATTERY_STATS → Reads battery level and temperature.  
- ACCESSWIFISTATE / ACCESSNETWORKSTATE → Provides WiFi and network details.  
- ACCESSFINELOCATION → Required for full SSID visibility.  
- INTERNET → Opens premium links and minimal external connections.  
- REQUESTIGNOREBATTERY_OPTIMIZATIONS → Allows exclusion from battery saver.  
- PACKAGEUSAGESTATS → Optional, for app usage statistics (requires manual activation).  


📦 Installation
1. Download the latest release from the Releases section.  
2. Install the APK on your Android device (Android 8+).  
3. Grant overlay and usage permissions in system settings for full functionality.
